Staff User Guide

Everything you need to use the Jarz POS system

Functions

🔑
1. Login & Profile
Sign in, choose Employee mode, change password, log out.
🕐
2. Shift Start & End
Open your shift with cash drawer opening balance; close it with actual cash at end of day.
🛒
3. POS Order Entry
Select customer first, add items, pick delivery slot or pickup, choose payment, submit.
📋
4. Kanban Board
Track orders through Recieved/Received → In Progress → Ready → Out for Delivery → Delivered → Cancelled.
🚚
5. Delivery Trips
Create delivery trips, assign courier, send out, and mark delivered.
💳
6. Daily Expenses
Record a daily expense and submit it for manager approval.
⚖️
7. Courier Balances
View outstanding cash balances owed to or by each courier.

A Typical Staff Day

  1. 1Login → choose Employee mode
  2. 2Start Shift → enter opening cash balance
  3. 3POS Order Entry → take and submit customer orders throughout the day
  4. 4Kanban Board → confirm orders, manage delivery status, record payments
  5. 5Delivery Trips → create trips for out-for-delivery orders
  6. 6Expenses → log any cash expenses incurred during the day
  7. 7End Shift → count cash, enter closing balance, submit
